# Portfólio Luiza Menezes

Projeto estático em HTML, CSS e JavaScript puro, pronto para publicar no GitHub Pages.

## Estrutura

- `index.html`: página principal do portfólio
- `styles.css`: estilos globais, responsividade e tema claro/escuro
- `script.js`: menu mobile, particles, tema, modais e lightbox
- `assets/`: imagens usadas no portfólio
- `projeto/*/index.html`: páginas dos estudos de caso
- `cv-luiza-menezes.pdf`: currículo linkado no site

## Publicação no GitHub Pages

1. Suba estes arquivos para o repositório.
2. Em Settings > Pages, selecione Deploy from a branch.
3. Escolha a branch principal e a pasta `/root`.

Não é necessário instalar dependências ou rodar build.
